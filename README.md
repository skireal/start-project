# Стартовая сборка с Gulp 4


## Установка

Требуются установленный [git](https://git-scm.com/) и [Node.js (LTS)](https://nodejs.org/en/).

1. Открыть терминал, попасть в папку проектов, клонировать этот репозиторий: `git clone https://github.com/skireal/start-project.git my-new-project` (где `my-new-project` — папка для нового проекта).
2. Перейти в папку нового проекта (в этом примере — `cd my-new-project`).
3. Удалить историю разработки: `rm -rf .git`
4. Установить зависимости проекта: `npm i`.


## Команды

```bash
npm start          # запуск сервера разработки (сборка БЕЗ библиотеки блоков)

```
